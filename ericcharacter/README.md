# Eric Character Assets

This directory contains illustrated character assets of Eric for use across the blog and other media.

## Base Character Description

The Eric character is a cartoon/comic-book style illustration of a developer in his element:

**Physical Appearance:**
- Man in his 30s-40s with short brown hair
- Athletic build
- Wearing a tan/olive "Jiu Jitsu" t-shirt
- Over-ear headphones

**Environment:**
- Multi-monitor tech workstation setup
- Screens displaying: code editor, trading charts, drone flight footage
- Raspberry Pi boards and drone components on desk
- Blue jiu-jitsu belt visible (hung or nearby)

**Background Elements:**
- Linux penguin (Tux) poster
- "Decentralize Everything" poster with chain links
- Rockwell Commander aircraft poster
- Bitcoin ATM

**Visual Style:**
- Comic book / cartoon illustration
- Clean lines with cel-shading
- Detailed but stylized
- Tech-forward color palette (blues, grays, accent colors)

## Usage Guidelines

When generating new character variants:
1. Maintain the same art style (comic book illustration)
2. Keep consistent facial features and build
3. Reference original image for accurate representation
4. Include at least one signature element (jiu-jitsu, tech, Bitcoin, aviation)
5. Use consistent color palette
